# CultureTranslate App

## Project Overview
CultureTranslate is a multi-lingual translation assistant designed to help travelers communicate across different cultures. It translates English phrases into selected languages and provides culturally appropriate etiquette tips.

## Key Features
- **Real-time Translation:** Connects to the MyMemory translation API.
- **Text-to-Speech (TTS):** Plays the translated text natively on your device.
- **Cultural Tips:** Displays custom etiquette advice according to the country selected.

## Technical Details
- **Languages Used:** Kotlin & XML Layouts
- **Libraries Used:** OkHttp, Android Material Components
